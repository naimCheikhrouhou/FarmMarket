# Farm maket Front-end steps
This file contains the steps and all the code used  for the React Native UI for Farm market

## Project Overview

The project is a business app named FarmMarket that allows users to publish their (products,services,proprety.
The app has a home screen that displays the current date, a grid of macro cards showing the total macros for the day, and a list of recent meals. There is also a screen for adding new meals and a screen for viewing all meals.
